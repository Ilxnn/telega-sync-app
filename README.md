# Утилита синхронизации Auto-Buyer Sync

Данная десктоп-утилита предназначена для автоматизации процесса синхронизации данных о рекламных закупках. Программа импортирует информацию из выгрузок Telega.in и CRM, после чего консолидирует ее в итоговой Google Таблице. Это позволяет исключить ручные операции по переносу данных и UTM-меток.

## Функционал

*   Программа импортирует данные из стандартных файлов формата `.xlsx`, выгруженных из Telega.in и CRM.
*   Она самостоятельно извлекает `utm_campaign`, дату размещения, стоимость и название канала.
*   Утилита связывает каждую закупку с соответствующим количеством лидов из CRM по UTM-метке.
*   При синхронизации с Google Sheets программа обновляет существующие записи и добавляет новые в конец таблицы, не нарушая структуру и формулы.
*   Предусмотрена возможность фильтрации закупок по конкретному исполнителю (Ваня, Глеб, Юра).
*   Реализована специальная логика обработки. Новые каналы исполнителя "Юра" добавляются с нулевым количеством лидов, а данные по лидам для его существующих записей не изменяются.
*   Программа не затрагивает записи о закупках, которые запланированы на текущую или будущие даты.

## Установка и настройка

Для первого запуска утилиты потребуется произвести единоразовую настройку окружения. Процесс занимает около 10-15 минут.

### Шаг 1. Установка Python

Для работы утилиты требуется стабильная версия Python.

1.  Необходимо скачать установщик **Python 3.12** с официального сайта по следующей ссылке.
    *   [**Ссылка для Windows (64-bit)**](https://www.python.org/ftp/python/3.12.3/python-3.12.3-amd64.exe)
2.  В процессе установки **обязательно** следует активировать опцию **`Add python.exe to PATH`** в первом окне установщика.
3.  Далее необходимо следовать инструкциям инсталлятора до завершения процесса.

### Шаг 2. Загрузка утилиты и установка зависимостей

1.  Скачайте исходный код с данного репозитория, нажав на зеленую кнопку **`< > Code`**, а затем **`Download ZIP`**.
2.  Распакуйте полученный архив в рабочий каталог (например, в папку на Рабочем столе).
3.  Запустите командную строку. Для этого нажмите `Win + R`, введите `cmd` и нажмите Enter.
4.  В командной строке перейдите в каталог с программой, используя команду `cd`. Например:
    ```cmd
    cd C:\Users\Имя_Пользователя\Desktop\telega-sync-app-main
    ```
5.  Выполните команду для установки всех требуемых библиотек.
    ```cmd
    pip install -r requirements.txt
    ```

### Шаг 3. Настройка доступа к Google API

Для предоставления утилите прав на редактирование Google Таблицы необходимо создать сервисный аккаунт.

1.  **Активация API**
    *   Перейдите в [Google Cloud Console](https://console.cloud.google.com/) и авторизуйтесь.
    *   Создайте новый проект с произвольным названием.
    *   Используя строку поиска, найдите `Google Sheets API`, перейдите на страницу API и активируйте его, нажав **`Enable`**.
    *   Повторите процедуру для `Google Drive API`.

2.  **Создание сервисного аккаунта**
    *   В меню навигации перейдите в раздел **`IAM & Admin` -> `Service Accounts`**.
    *   Нажмите **`+ CREATE SERVICE ACCOUNT`**.
    *   Укажите имя аккаунта (например, `sync-bot`), нажмите `CREATE AND CONTINUE`, а затем `DONE`.

3.  **Получение ключа в формате JSON**
    *   В списке сервисных аккаунтов выберите только что созданный.
    *   Перейдите на вкладку **`KEYS`**.
    *   Нажмите **`ADD KEY` -> `Create new key`**.
    *   Выберите тип **`JSON`** и нажмите `CREATE`.
    *   Сохраните скачанный файл. **Переименуйте его в `credentials.json`** и поместите в тот же каталог, где находится файл `app.py`.

4.  **Предоставление доступа к таблице**
    *   Откройте файл `credentials.json` в текстовом редакторе и скопируйте email-адрес из поля `"client_email"`.
    *   Откройте целевую Google Таблицу.
    *   Нажмите на кнопку **`Настройки доступа` (Share)**.
    *   Вставьте скопированный email-адрес в поле для добавления пользователей и предоставьте ему права **`Редактора` (Editor)**.

**На этом первоначальная настройка завершена.**

## Порядок работы

1.  Убедитесь, что файл `credentials.json` находится в одном каталоге с `app.py`.
2.  Запустите утилиту, выполнив двойной щелчок по файлу `app.py`.
3.  При первом запуске откроется окно начальной настройки. В нем потребуется указать ссылку на Google Таблицу и название рабочего листа. Эти данные сохранятся в файле `config.json` для последующих запусков. Изменить их можно в любой момент через кнопку "Настройки" в главном окне.
4.  В главном окне программы выберите файлы выгрузок из Telega.in и CRM.
5.  Выберите исполнителя из выпадающего списка для фильтрации данных.
6.  Нажмите кнопку "Синхронизировать".
7.  Процесс выполнения будет отображаться в лог-панели. По окончании появится системное уведомление об успешном завершении.
