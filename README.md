# Auto-Buyer Sync

Десктоп-утилита для автоматической синхронизации рекламных закупок из Telega.in и данных по лидам из CRM с итоговой Google Таблицей. Экономит часы рутинной работы по копированию данных и UTM-меток.

## ✅ Основные возможности

*   **Импорт данных:** Читает стандартные `.xlsx` выгрузки из Telega.in и вашей CRM.
*   **Автоматический парсинг:** Самостоятельно извлекает `utm_campaign` из ссылок, находит дату, цену и название канала.
*   **Сопоставление лидов:** Связывает каждую закупку с количеством лидов из выгрузки CRM по UTM-метке.
*   **Умная синхронизация с Google Sheets:**
    *   **Обновляет** данные (дату, цену, лиды) для уже существующих в таблице каналов.
    *   **Добавляет** новые каналы в конец таблицы, не ломая существующие строки и формулы.
*   **Фильтрация:** Позволяет обрабатывать закупки только по конкретному байеру (Ваня, Глеб, Юра).
*   **Простой интерфейс:** Интуитивно понятное окно приложения для Windows.

## ⚙️ Установка и первая настройка

Чтобы запустить утилиту, нужно один раз подготовить компьютер. Это займет 10-15 минут.

### Шаг 1: Установка Python

Для работы скрипта нужен Python. Важно установить правильную, стабильную версию.

1.  Скачайте установщик **Python 3.12** с официального сайта:
    *   [**Прямая ссылка на установщик для Windows (64-bit)**](https://www.python.org/ftp/python/3.12.3/python-3.12.3-amd64.exe)
2.  Запустите установщик. В самом первом окне **ОБЯЗАТЕЛЬНО** поставьте галочку **`Add python.exe to PATH`**. Это критически важно!
3.  Далее просто нажимайте `Install Now` и `Next`, пока установка не завершится.

### Шаг 2: Скачивание утилиты и установка библиотек

1.  Скачайте код утилиты с этого репозитория. Нажмите зеленую кнопку **`< > Code`** -> **`Download ZIP`**.
2.  Распакуйте скачанный архив в удобную для вас папку (например, на Рабочий стол в папку `Telega Sync`).
3.  Откройте командную строку: нажмите `Win + R`, введите `cmd` и нажмите Enter.
4.  Перейдите в папку с программой с помощью команды `cd`. Например:
    ```cmd
    cd C:\Users\Имя_Пользователя\Desktop\Telega Sync
    ```
5.  Выполните одну команду, которая автоматически установит все необходимые компоненты:
    ```cmd
    pip install -r requirements.txt
    ```

### Шаг 3: Настройка доступа к Google API (самый важный шаг!)

Чтобы скрипт мог редактировать вашу Google Таблицу, нужно создать для него специальный "ключ-доступ" (сервисный аккаунт).

1.  **Включение API:**
    *   Перейдите в [Google Cloud Console](https://console.cloud.google.com/). Войдите под своим Google-аккаунтом.
    *   Создайте новый проект (любое название).
    *   В строке поиска наверху введите `Google Sheets API`, перейдите на страницу API и нажмите **`Enable` (Включить)**.
    *   Вернитесь в поиск, введите `Google Drive API` и также нажмите **`Enable`**.

2.  **Создание сервисного аккаунта:**
    *   В меню слева перейдите в **`IAM & Admin` -> `Service Accounts`** (IAM и администрирование -> Сервисные аккаунты).
    *   Нажмите **`+ CREATE SERVICE ACCOUNT`**.
    *   Придумайте имя (например, `telega-sync-bot`) и нажмите `CREATE AND CONTINUE`, затем `DONE`.

3.  **Получение JSON-ключа:**
    *   Вы окажетесь в списке аккаунтов. Нажмите на email только что созданного аккаунта.
    *   Перейдите на вкладку **`KEYS` (Ключи)**.
    *   Нажмите **`ADD KEY` -> `Create new key`**.
    *   Выберите тип **`JSON`** и нажмите `CREATE`.
    *   Браузер скачает файл. **Переименуйте его в `credentials.json`** и положите в ту же папку, где лежит файл `app.py`.

4.  **Предоставление доступа к таблице:**
    *   Откройте `credentials.json` блокнотом. Скопируйте оттуда email-адрес, указанный в поле `"client_email"`. Он выглядит примерно так: `telega-sync-bot@имя-проекта.iam.gserviceaccount.com`.
    *   Откройте вашу рабочую Google Таблицу.
    *   Нажмите **`Настройки доступа` (Share)** в правом верхнем углу.
    *   Вставьте скопированный email в поле для добавления пользователей и выдайте ему права **`Редактора` (Editor)**.

### Шаг 4: Настройка ссылки на вашу таблицу в скрипте

1.  Откройте файл `app.py` в любом текстовом редакторе (например, Блокнот или VS Code).
2.  Найдите строку ~25 и вставьте ссылку на **вашу** Google Таблицу:
    ```python
    self.sheet_url = "https://docs.google.com/spreadsheets/d/ТВОЙ_ID_ТАБЛИЦЫ/edit" 
    ```
3.  Сохраните файл.

**На этом установка завершена!**

## 📖 Как пользоваться

1.  Убедитесь, что файл `credentials.json` лежит в одной папке с `app.py`.
2.  Запустите утилиту, дважды кликнув по файлу `app.py`.
    > *Если ничего не происходит, запустите через `cmd`: введите `python app.py` и нажмите Enter.*
3.  В открывшемся окне:
    *   Нажмите **"Обзор"** и выберите скачанный `.xlsx` файл-выгрузку из **Telega.in**.
    *   Нажмите второй **"Обзор"** и выберите `.xlsx` файл-выгрузку из **CRM**.
    *   Выберите из выпадающего списка, закупки какого байера нужно обработать.
4.  Нажмите большую кнопку **"работать быстр..."**.
5.  Следите за логами в нижней части окна. По завершении появится сообщение об успехе.
6.  Проверьте вашу Google Таблицу — данные должны были обновиться.
